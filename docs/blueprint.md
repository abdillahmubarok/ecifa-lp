# **App Name**: Ecifa.id Landing Page

## Core Features:

- Hero Section: Display a Hero Section with headline, sub-headline, background visual, and CTAs (Pelajari Lebih Lanjut, Lihat Riset Kami).
- About Us: Present an About Us section detailing Ecifa.id's vision, mission (Riset multidisiplin, Teknologi pembelajaran, Kolaborasi strategis, Penyebaran hasil riset), and background.
- Research Focus: Showcase research areas (Kurikulum & Pedagogi, EdTech, Kebijakan & Evaluasi Pendidikan, Pengembangan Profesional) in a card layout with icons.
- Strengths: Highlight Ecifa.id's strengths (Berbasis Data, Tim Ahli, Fokus Indonesia, Jaringan Luas) with icons.
- Publications: Feature latest publications (cover, title, summary, link) and filter them by publication date
- Collaboration CTA: Include a collaboration CTA with a message and a 'Hubungi Kami' button linked to kolaborasi@ecifa.id
- Footer: Implement a footer with logo, navigation, contact info (email info@ecifa.id, kolaborasi@ecifa.id), social media links, and a copyright notice.

## Style Guidelines:

- Primary color: Dark blue (#0A2B4C), reflecting professionalism and trust.
- Background color: Very light blue (#EBF4FA), creating a clean and calm base for the content.
- Accent color: Green (#28A745), suggesting growth, innovation and a reference to nature/education. Use this as an accent throughout the design to draw the user's eye to the primary actions and information on the page.
- Font recommendation: 'Poppins' sans-serif for headlines and body text; offers a modern, clean, and readable aesthetic.
- Responsive design: Ensure the website adapts smoothly to different screen sizes.
- Use a consistent style of simple, modern icons related to education and research, supporting visual communication.
